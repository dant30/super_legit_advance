# Super Legit Advance - Loan Management System

## Overview
A comprehensive loan management system for microfinance institutions with M-Pesa payment integration and SMS notifications.

## Features
- Loan application and management
- Customer management
- Repayment tracking
- M-Pesa payment integration
- SMS notifications
- Reporting and analytics
- Audit logging
- Role-based access control

## Tech Stack
- **Backend**: Django + Django REST Framework
- **Frontend**: React + TypeScript + Vite + Tailwind CSS
- **Database**: PostgreSQL
- **Cache**: Redis
- **Task Queue**: Celery
- **SMS**: Africa's Talking API
- **Payments**: M-Pesa API

## Quick Start

### Prerequisites
- Python 3.11+
- Node.js 18+
- Docker & Docker Compose
- PostgreSQL 15+
- Redis 7+

### Development Setup
1. Clone the repository
2. Copy `.env.example` to `.env` and configure
3. Run: `docker-compose up`
4. Backend: http://localhost:8000
5. Frontend: http://localhost:3000

### API Documentation
Access API docs at: http://localhost:8000/api/docs/
